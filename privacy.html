隐私政策

最后更新日期：2026 年 3 月 18 日

虾忙 BuzyClaw（以下简称"本应用"）由 BuzyClaw 团队开发和运营。我们非常重视用户的隐私保护，本隐私政策旨在说明我们如何收集、使用和保护您的信息。

1. 信息收集

本应用在运行过程中可能收集以下类型的信息：

  • 设备信息：操作系统版本、设备型号等基础信息，用于优化应用兼容性。
  • iCloud 标识：用于 macOS 与 iOS 设备之间的配对连接，我们不会存储您的 Apple ID。
  • OpenClaw 服务数据：您的 Agent 配置、任务记录等数据存储在您自己的 OpenClaw 服务端，本应用仅作为客户端进行显示和整理。

2. 信息使用

我们收集的信息仅用于以下目的：

  • 提供和维护应用的核心功能
  • 实现 macOS 与 iOS 设备间的无缝连接
  • 改善应用性能和用户体验
  • 排查技术故障和提供技术支持

3. 数据存储与安全

本应用采用端到端通信方式，您的 Agent 数据直接存储在您自己控制的 OpenClaw 服务端。macOS 与 iOS 设备间的通信通过 iCloud 进行端到端直连，我们不会在其他第三方服务器上存储您的业务数据。

4. 第三方服务

本应用可能使用以下第三方服务：

  • Apple iCloud：用于设备间的数据同步与配对连接。
  • OpenClaw 平台：作为 AI Agent 的运行和管理后端。

这些第三方服务有各自的隐私政策，我们建议您查阅相关条款。

5. 信息共享

我们不会向任何第三方出售、交易或以其他方式转让您的个人信息，除非：

  • 获得您的明确授权同意
  • 法律法规要求或政府机关依法要求
  • 为保护我们或其他用户的合法权益

6. 用户权利

您有权随时：

  • 查看和管理应用内存储的数据
  • 删除应用及其所有本地数据
  • 断开 macOS 与 iOS 设备的连接
  • 联系我们要求删除与您相关的信息

7. 未成年人保护

本应用不面向 13 岁以下的未成年人。我们不会故意收集 13 岁以下未成年人的个人信息。

8. 隐私政策的更新

我们可能会不定期更新本隐私政策。更新后的政策将在本页面发布，并更新"最后更新日期"。建议您定期查看本页面以了解最新的隐私保护措施。

9. 联系我们

如果您对本隐私政策有任何疑问或建议，请通过我们的官方网站联系我们。
