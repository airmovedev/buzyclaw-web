1. 服务说明

本应用是基于 OpenClaw 平台的原生客户端，提供 AI Agent 的可视化管理功能，包括但不限于：

  • Agent 管理与监控
  • 任务创建与追踪
  • 记忆数据管理
  • 定时任务配置
  • macOS 与 iOS 设备间的协同操作

2. 使用要求

使用本应用需要满足以下条件：

  • macOS 版本：macOS 14.0 (Sonoma) 或更高版本
  • iOS 版本：iOS 17.0 或更高版本
  • 拥有可用的 OpenClaw 服务端
